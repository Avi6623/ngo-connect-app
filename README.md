# NGOConnect

NGOConnect is the public-facing platform in this NGO ecosystem. It is built for donors, volunteers, and community visitors who want to explore NGO activities, join events, contribute to campaigns, and track where support is being mapped.

## What It Includes

- Public campaign discovery with progress visibility
- Event exploration and participation flow
- Volunteer registration flow
- Donation participation flow
- Donor tracking view to understand where money is being routed
- Privacy-safe public transparency feed

## Tech Stack

- React
- Vite
- Tailwind CSS
- TypeScript
- Shared Node.js + Express backend
- Local JSON storage via backend APIs

## Local Run

```bash
cd "/Users/main/NGO Apps/ngo-connect"
npm install
npm run dev
```

Default local URL: `http://localhost:5174`

Expected backend: `http://localhost:4000`
