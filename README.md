# NGO Connect App

A full-stack web application built to help NGOs manage volunteers, cases, and community activities efficiently.

## Tech Stack
- Frontend: React / React Native
- Backend: Node.js
- Database: MongoDB
- Tools: GitHub, Postman

## Features
- Volunteer registration and management
- Case tracking for NGOs
- Clean and modular project structure
- API integration and testing

## My Role

- Worked on frontend components and backend logic
- Integrated APIs and tested them using Postman
- Debugged issues and improved overall flow
- Used modern tools including AI to speed up development while understanding the core logic

## Project Status
- In progress (academic learning project)
- More features and improvements planned

## How to Run (Optional)
1. Clone the repository
2. Install dependencies using `npm install`
3. Start the app using `npm start`
